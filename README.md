# Home Server Lab

This project documents the process of building a private cloud and cybersecurity lab from a repurposed PC.

## Stack
- Ubuntu Server 24.04
- Docker & Docker Compose
- Nextcloud + MariaDB
- Cloudflare Tunnel
- SSH Key Authentication
- Rclone Backup

## Features
- Private cloud storage accessible globally via HTTPS
- No port forwarding on home router
- Headless remote management
- Automated backup to Google Drive

## Domain
https://cloud.ardana-cloud.my.id

## Purpose
This repository serves as a technical portfolio for Red Team / Ethical Hacking career path.
